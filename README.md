# TinkoffTicker
Плагин находит на страницах тикеры и делает их ссылками на "Тинькофф Брокер"<br>
К тикерам добавляются ссылки на сайтах:
* finance.yahoo.com
* finviz.com
* investing.com
* tradingview.com
<br>
